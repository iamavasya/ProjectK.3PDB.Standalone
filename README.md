# База третьопробників
![Static Badge](https://img.shields.io/badge/Code_name-ProjectK.3PDB.Standalone-orange)
![GitHub Release](https://img.shields.io/github/v/release/iamavasya/ProjectK.3PDB.Standalone)

## Що це?
Це вузькоспеціалізований web-додаток, загорнутий у desktop, створений для організації Пласт, що має на меті спростити роботу певним ролям у організації.

## Що це? (для пластунів)
Це додаток для Windows, що дає змогу референтам 3-ї проби легше вести облік 3-пробників у своїх осередках.

## Функції
- Зручний облік третьопробників;
- Статус проби;
- Лічильник днів до закриття;
- Імпорт та експорт даних;
- Пошук по базі.
